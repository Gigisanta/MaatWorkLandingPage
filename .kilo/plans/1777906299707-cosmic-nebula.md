# Cosmic Nebula Background - Implementation Plan

## Problem Analysis

The user reports nothing displays in the browser with no console errors. This suggests the React Three Fiber Canvas may be failing silently due to:
1. WebGL context issues
2. SSR hydration problems with dynamic import
3. Missing error boundary wrapper on the galaxy component

## Solution Approach

Create a **standalone HTML file** with pure vanilla JavaScript and CSS animations that:
- Works immediately when opened in any browser
- Does not depend on React/Three.js
- Achieves 60fps with optimized requestAnimationFrame
- Implements all requested visual effects

---

## Implementation Details

### 1. HTML Structure
```
- Container (full viewport, fixed position)
  - Nebula gradient layer (CSS radial/linear gradients)
  - Stars canvas layer (canvas 2D for performance)
  - Particles canvas layer (canvas 2D for particles)
  - Central content area
```

### 2. Stars System (Canvas 2D)
- **Count**: 400-600 stars (responsive to screen size)
- **Twinkle effect**: Sinusoidal opacity oscillation with randomized phase/frequency
- **Parallax**: Mouse position influences star layer movement (subtle, 0.02-0.05 multiplier)
- **Colors**: White, pale blue (#a5b4fc), pale purple (#c4b5fd), pale pink (#f9a8d4)
- **Sizes**: 0.5px to 2.5px with glow effect for larger stars

### 3. Floating Particles (requestAnimationFrame)
- **Count**: 80-120 particles
- **Movement**: Perlin-noise-like organic motion using simplex noise approximation
- **Speed**: 0.2-0.8px per frame (varied)
- **Size**: 1px to 4px with soft glow
- **Colors**: Gradient from purple (#8b5cf6) to pink (#ec4899) to blue (#3b82f6)
- **Parallax**: Mouse tracking with 0.03-0.08 multiplier (faster than stars)
- **Opacity**: 0.3-0.7 with breathing effect

### 4. Nebula Background (CSS)
- Multiple layered radial gradients
- Colors: Deep purple (#1e1b4b), violet (#4c1d95), magenta (#a21caf), blue (#1e3a8a)
- Positioned at different areas creating depth
- Subtle animation: slow drift/rotation using CSS transforms
- Blend mode: screen/lighten for cosmic glow

### 5. Performance Optimizations
- Use `will-change: transform` on animated layers
- Throttle mouse move events (requestAnimationFrame alignment)
- Use `devicePixelRatio` capped at 2 for HiDPI displays
- Implement visibility API to pause when tab hidden
- Canvas clearing with `fillRect` instead of `clearRect` for trail effects

### 6. Responsive Design
- Stars count scales with viewport (min 200, max 800)
- Particles count scales with viewport (min 50, max 150)
- Nebula gradient sizes use vw/vh units
- Touch support for parallax on mobile
